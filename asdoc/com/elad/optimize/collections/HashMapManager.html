Classpublic class HashMapManager
InheritanceHashMapManager HashMapCollection Object

A Singleton utility class to capture global variables for flex and pure Action script and store them for future needs. The class extends the HashMapCollection API, which allow creation of pairs of keys and values

setFlashVars()method 
public function setFlashVars(parameters:Object):void

Method to set the flash vars variables

Parameters

parameters:Object


Example
The following example sets the flash vars in Flex 
                 private var flashvar:GlobalVariables = GlobalVariables.getInstance();
                 flashvar.setFlashVars(Application.application.parameters); 
                 var str:String = flashvar.getItemValue("configUrl");
The following example sets the flash vars in Pure Action Script 
                 private var flashvar:GlobalVariables = GlobalVariables.getInstance();
                 flashvar.setFlashVars(LoaderInfo(this.root.loaderInfo).parameters); 
                 var str:String = flashvar.getItemValue("configUrl");
